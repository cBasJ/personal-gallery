# Personal Gallery

React + TypeScript personal portfolio and resume website.

## Development

```bash
npm install
npm run dev
```

Local site:

```text
http://localhost:5173/
```

## Build

```bash
npm run build
```

## Edit Content

Most resume and portfolio content lives in:

```text
src/profile.ts
```

Fill out `content-brief.md` first, then copy the final content into `src/profile.ts`.

## Current Sections

- Hero profile summary
- Key metrics
- Strengths
- Selected projects
- Work experience
- Capabilities
- Contact links
